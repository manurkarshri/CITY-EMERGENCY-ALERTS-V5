# Contributing to CITY EMERGENCY ALERTS

Thank you for your interest in improving CITY EMERGENCY ALERTS.

This project aims to provide free and accessible emergency awareness information for the community.

---

## Ways to Contribute

You can help by:

- Reporting bugs
- Suggesting useful emergency features
- Adding official data sources
- Improving documentation
- Improving UI/UX
- Fixing broken links
- Improving performance
- Improving accessibility
- Translating content
- Adding locality information

---

## Before Opening an Issue

Please check that the issue has not already been reported.

Include:

- Browser
- Device
- Screenshots
- Steps to reproduce
- Expected behaviour
- Actual behaviour

---

## Pull Requests

Please ensure:

- Code is well documented.
- Existing functionality is not broken.
- No paid APIs are introduced.
- Public data sources remain free.
- GitHub Pages compatibility is maintained.
- Offline functionality continues to work.

---

## Coding Standards

- Use clear variable names.
- Keep functions modular.
- Minimize external dependencies.
- Prefer vanilla JavaScript.
- Keep the interface simple.
- Optimize for mobile devices.

---

## Community Contributions

Suggestions for:

- New official emergency sources
- River monitoring
- Weather improvements
- Traffic intelligence
- Emergency preparedness

are always welcome.
