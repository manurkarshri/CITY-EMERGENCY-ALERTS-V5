Production-oriented, free, GitHub Pages compatible emergency intelligence dashboard for Pune and nearby areas.

- 50+ project files
- 10 GitHub Actions workflows
- 20+ free live data/search sources through Google News RSS and public endpoints
- 100+ locality and alias mappings
- Advanced deduplication engine
- Incident correlation engine
- Weather intelligence
- River/flood intelligence framework
- Traffic intelligence framework
- Official alert aggregation
- Automatic expiry and cleanup
- PWA installation
- Offline mode
- Production GitHub Pages deployment support
- No paid APIs

## Deployment

1. Replace the files in your repository with the contents of this ZIP.
2. Commit all files to `main`.
3. Go to **Actions** and enable workflows if GitHub asks.
4. Run **V5 Update All Emergency Data** manually once.
5. Go to **Settings → Pages**.
6. Select **Deploy from branch → main → /(root)**.
7. Open your GitHub Pages URL.
