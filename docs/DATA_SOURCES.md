# Data Sources

This V5 package uses free public sources and public web feeds.

## Main sources
- Google News RSS English
- Google News RSS Marathi
- IMD related search
- PMC related search
- PCMC related search
- Pune Police / PCMC Police related search
- MSEDCL related search
- River/flood related search
- Open-Meteo free weather API

No paid APIs are required.
